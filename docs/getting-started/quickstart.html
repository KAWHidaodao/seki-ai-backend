Quick Start

Get Seki Agent running in under 5 minutes. From zero to a self-evolving on-chain agent.

Prerequisites

  • Node.js 18+ or Python 3.10+
  • A BSC or Base RPC endpoint (Alchemy, QuickNode, or public)
  • Binance API key (optional — for CEX trading skill)

Install

Terminal
$ npm install -g @seki/agent

# or with pip
$ pip install seki-agent

Initialize

Create a new Seki agent workspace. This sets up the memory system, skill registry, and tool configuration.

Terminal
$ seki init --chain bsc

Initializing Seki Agent v0.8.3...
Creating workspace at ~/.seki/
├── memory/          → Persistent memory store
├── skills/          → Skill modules
├── tools/           → Tool configurations
├── strategies/      → Evolved strategies
└── config.yaml      → Agent configuration

✓ Workspace initialized
✓ Memory system ready
✓ Default skills loaded (3)
✓ BSC RPC connected

Configure

Edit ~/.seki/config.yaml to set your chain, API keys, and enabled skills.

~/.seki/config.yaml
chain: bsc
rpc: https://bsc-dataseed1.binance.org

skills:
  - meme-launch
  - cex-trading
  - market-sense

tools:
  binance:
    api_key: ${BINANCE_API_KEY}
    secret: ${BINANCE_SECRET}
    sandbox: true   # use testnet first

evolution:
  enabled: true
  cycles_per_hour: 60
  memory_consolidation: every_100_cycles

Run

Terminal
$ seki start

[SEKI] Agent online
[SEKI] Evolution loop started — cycle #1
[SEKI] Observing: 14,283 pending transactions
[SEKI] Pattern detected: whale_accumulation
[SEKI] Strategy pool: 3 active, 0 deprecated
[SEKI] Listening for signals...
Tip: Use seki start --dry-run to observe and learn without executing any trades. The agent still evolves — it just doesn't act on the market.

What Happens Next?

Once running, Seki enters the Evolution Loop:

  1. Observe — Ingests on-chain data, price feeds, and social signals
  2. Learn — Detects patterns, builds internal market models
  3. Evolve — Mutates strategies, promotes winners, prunes losers
  4. Act — Executes trades or token launches, then feeds results back

Each cycle makes the agent smarter. Read more in Self-Evolution.
