Installation

Platform-specific installation guides and system requirements.

System Requirements

ComponentMinimumRecommended
CPU2 cores4+ cores
RAM2 GB8 GB
Disk500 MB2 GB (for memory store)
Node.js18.0+20 LTS
Python3.10+3.12+
NetworkStable connectionLow-latency RPC

NPM (Recommended)

Terminal
$ npm install -g @seki/agent

# verify installation
$ seki --version
seki-agent v0.8.3

Python (pip)

Terminal
$ pip install seki-agent

# or with uv for faster install
$ uv pip install seki-agent

Docker

Terminal
$ docker pull sekiagent/seki:latest
$ docker run -d \
    --name seki \
    -v ~/.seki:/root/.seki \
    -e BINANCE_API_KEY=$BINANCE_API_KEY \
    -e BINANCE_SECRET=$BINANCE_SECRET \
    sekiagent/seki:latest

From Source

Terminal
$ git clone https://github.com/seki-agent/seki.git
$ cd seki
$ npm install
$ npm link

# now available globally
$ seki --version

Chain-Specific Setup

BSC (BNB Smart Chain)

Default chain. Uses public BSC RPC by default, but a dedicated endpoint is recommended for production.

Terminal
$ seki init --chain bsc --rpc https://your-bsc-rpc.com

Base

Terminal
$ seki init --chain base --rpc https://your-base-rpc.com
Note: Multi-chain support is on the roadmap. Currently, Seki runs one chain per instance. Run multiple instances for multi-chain coverage.
