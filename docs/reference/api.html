API Reference

REST API endpoints for monitoring, controlling, and integrating with your Seki agent.

Base URL

Default
http://localhost:3000/api/v1

Authentication

All endpoints require a Bearer token. Generate one via the CLI:

Token Setup
seki auth token --generate

# Use in requests:
curl -H "Authorization: Bearer sk_live_..." http://localhost:3000/api/v1/status

Agent

GET /status

Returns agent health, uptime, and current state.

Response 200
{
  "status": "running",
  "uptime": "14d 6h 32m",
  "version": "0.8.2",
  "generation": 848,
  "active_skills": 12,
  "active_tools": 7,
  "memory_usage": "1.2 GB",
  "events_per_sec": 847
}

POST /pause

Pause all agent activity. Active trades are held, no new positions opened.

POST /resume

Resume agent activity after pause.

POST /emergency-stop

Immediately halt all operations and close open positions.

Warning: Emergency stop will market-sell all open positions. Use only in critical situations.

Skills

GET /skills

List all installed skills with status and fitness scores.

Response 200
{
  "skills": [
    {
      "name": "meme-launch",
      "version": "2.3.1",
      "status": "active",
      "fitness": 0.84,
      "executions_24h": 47,
      "last_triggered": "2025-01-15T10:32:00Z"
    },
    {
      "name": "cex-trading",
      "version": "3.1.0",
      "status": "active",
      "fitness": 0.79,
      "executions_24h": 312,
      "last_triggered": "2025-01-15T10:34:12Z"
    }
  ]
}

GET /skills/:name

Get detailed information about a specific skill.

POST /skills/:name/enable

Enable a disabled skill.

POST /skills/:name/disable

Disable a skill. Running executions complete, no new triggers processed.

PUT /skills/:name/config

Update skill configuration parameters.

Request Body
{
  "max_position": 1000,
  "stop_loss": 0.03,
  "entry_threshold": 0.85
}

Memory

GET /memory/stats

Memory system statistics.

Response 200
{
  "short_term": { "entries": 1247, "size": "48 MB", "window": "24h" },
  "long_term": { "entries": 8923, "size": "312 MB", "oldest": "2024-11-01" },
  "episodic": { "episodes": 2341, "size": "856 MB" }
}

GET /memory/search

Search memory with semantic query.

Query Parameters
?q=large+whale+transfer&tier=all&limit=20&from=2025-01-01

DELETE /memory/short-term

Clear short-term memory buffer.

Evolution

GET /evolution/status

Current evolution engine state.

Response 200
{
  "generation": 848,
  "avg_fitness": 0.71,
  "best_fitness": 0.91,
  "active_strategies": 12,
  "total_pruned": 47,
  "mutations_today": 23,
  "mutation_success_rate": 0.61
}

GET /evolution/history

Evolution history with fitness progression.

Query Parameters
?generations=50&include_mutations=true

POST /evolution/force-cycle

Trigger an immediate evolution cycle outside the normal schedule.

Tools

GET /tools

List registered tools with health status.

GET /tools/:name/health

Detailed health check for a specific tool.

Response 200
{
  "name": "binance",
  "status": "healthy",
  "latency_ms": 42,
  "uptime_pct": 99.97,
  "last_check": "2025-01-15T10:35:00Z",
  "rate_limit": { "used": 340, "limit": 1200, "reset_in": "38s" }
}

POST /tools/:name/reconnect

Force reconnect a tool (useful after credential rotation).

WebSocket

Real-time event stream for dashboards and integrations.

WebSocket Connection
ws://localhost:3000/ws?token=sk_live_...

// Subscribe to channels
{ "subscribe": ["events", "trades", "evolution", "alerts"] }

// Event format
{
  "channel": "trades",
  "type": "order_filled",
  "data": {
    "skill": "meme-launch",
    "pair": "PEPE/USDT",
    "side": "buy",
    "amount": 500,
    "price": 0.00001234
  },
  "timestamp": "2025-01-15T10:36:00Z"
}

Error Responses

Error Format
{
  "error": {
    "code": "SKILL_NOT_FOUND",
    "message": "Skill 'xyz' is not installed",
    "status": 404
  }
}
CodeStatusDescription
UNAUTHORIZED401Missing or invalid token
FORBIDDEN403Insufficient permissions
NOT_FOUND404Resource not found
AGENT_PAUSED409Agent is paused, action not available
RATE_LIMITED429Too many requests
INTERNAL500Internal server error
