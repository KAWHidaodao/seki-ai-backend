Skills Catalog

Built-in Skills

meme-launch

LIVE

Detects new meme token launches across DEXes, evaluates initial liquidity, social buzz, and contract safety. Executes early entry with configurable position sizing.

Version2.3.1
Triggerschain.pair_created, chain.liquidity_added
Toolsdexscreener, etherscan, twitter, web3-rpc
Fitness0.84

Parameters

ParamTypeDefaultDescription
min_liquiditynumber$10,000Minimum pool liquidity to consider
max_positionnumber$500Maximum per-trade position size
entry_thresholdnumber0.79Confidence score required to enter
take_profitnumber[][2x, 5x, 10x]Staged take-profit levels
stop_lossnumber-50%Maximum loss before exit

Community Skills

Browse and install community-contributed skills from the marketplace:

Marketplace Commands
# Search skills
seki skill search "arbitrage"

# View skill details
seki skill info cross-dex-arb

# Install from marketplace
seki skill install cross-dex-arb@1.2.0

# List installed skills
seki skill list --installed
