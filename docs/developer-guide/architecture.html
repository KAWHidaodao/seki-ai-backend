Core Components

Brain (Decision Engine)

The central coordinator. Receives processed signals, consults memory, evaluates available strategies, and dispatches actions through the skill engine.

  • Input: Processed events from observation pipeline
  • Processing: Multi-factor decision matrix with configurable weights
  • Output: Action signals to skill engine with priority and confidence

Memory System

Three-tier memory architecture providing context at different time scales. See Memory System for details.

Evolution Engine

Runs the observe-learn-evolve-act loop. Manages strategy populations, fitness scoring, and mutation cycles. See Self-Evolution for details.

Skill Engine

Loads, validates, and executes skill modules. Manages skill lifecycle and inter-skill communication. See Skill Engine for details.

Tool Registry

Manages authenticated connections to external services. See Tool Registry for details.

Data Flow

Request Lifecycle
External Event (e.g., token launch detected)
  │
  ├─→ Event Bus ─→ Observation Pipeline
  │                    │
  │                    ▼
  │              Pattern Matching
  │                    │
  │                    ▼
  │              Brain evaluates signal
  │                ├─ Query Memory (similar past events)
  │                ├─ Check active strategies
  │                └─ Score confidence
  │                    │
  │                    ▼
  │              Skill Dispatch (e.g., meme-launch)
  │                    │
  │                    ▼
  │              Tool Execution (e.g., binance.place_order)
  │                    │
  │                    ▼
  │              Result → Memory Store → Fitness Update
  │
  └─→ Evolution Engine processes outcome next cycle

Deployment

Docker Compose
services:
  seki:
    build: .
    environment:
      - NODE_ENV=production
      - REDIS_URL=redis://redis:6379
    volumes:
      - ./config:/app/config
      - ./skills:/app/skills
      - ./data:/app/data
    ports:
      - "3000:3000"
    depends_on:
      - redis

  redis:
    image: redis:7-alpine
    volumes:
      - redis-data:/data
Minimum Requirements: 2 vCPU, 4GB RAM, 20GB SSD. For full evolution with multiple concurrent strategies, recommend 4 vCPU and 8GB RAM.
